# 노션 전체 인벤토리(2026.07, 480항목) 델타 반영 (2026-07-21)

작성 배경: 사용자가 Notion 3.4(2026년 4월 릴리스) 기준 26개 카테고리 약 480개 항목의 전수 인벤토리를
제공. 요청은 "전부 구현"이었으나, 그대로는 로드맵·무료티어·ponytail(YAGNI) 원칙 위반이라 **로드맵
반영·계획화**로 방향 전환(사용자 승인, 2026-07-21).

이 문서는 **델타 전용**이다. 백지 재분석이 아니다 —
[notion-gap-analysis-2026-07-21.md](./notion-gap-analysis-2026-07-21.md)가 이미 노션 코어 26축을
LDD 로드맵에 매핑(Phase 배치·무료티어·1인 필터·로드맵 정합)했고, [../FEATURES.md](../FEATURES.md)가
146개 신규 기능을 Phase에 배정했다. 이 문서의 역할은 **그 두 문서가 쓰인 시점(2025 학습지식 기준)
이후·더 세분된 이번 인벤토리가 무엇을 추가/변경하는가만** 추리는 것이다.

**총괄 판정**: 480항목의 약 95%는 **이미 계획된 Phase에 흡수**되거나 **구조적으로 제외**된다.
진짜 신규 반영분은 (1) 2026 신기능 소수(대부분 소규모 또는 enterprise 제외), (2) LDD 차별화의
노션 무대응 재확인, (3) 무료티어·1인 부적합 대량 제외의 명문화. **로드맵 순서·Phase 정의 변경 없음**
(gap-analysis 8절 "Phase 순서 유지" 결정 유지).

## 1. 판정 필터 (gap-analysis 4절 + tier 필터)

1. **라이브러리 무료 제공** — BlockNote가 주는 블록/슬래시/드래그/마크다운은 따라간다(P9). CRDT
   동시편집처럼 인프라 비용이 큰 것은 버린다.
2. **1인 사용자에게 유의미** — 권한 체계·게스트·팀스페이스·People Directory·멤버 관리는 무의미 →
   제외.
3. **무료 티어 저촉** — Supabase 500MB/1GB, Gemini 무료 쿼터. 파일 업로드·임베딩은 용량 정책과
   함께 설계.
4. **플랜 tier(신규 필터)** — 인벤토리의 (B)=Business/(E)=Enterprise 표기 항목은 유료 인프라가
   전제라 **무료 원칙상 구현 불가** → 구조적 제외(카테고리 16·17 대부분, 커넥터 B/E, 행단위 권한 등).

## 4. LDD 차별화의 노션 무대응 (재확인 — 이 인벤토리에도 대응물 없음)

인벤토리 480항목 어디에도 아래 대응물이 없다. LDD의 방어 가능한 포지션이 재확인된다(gap-analysis 3절).

- **3D 오리 + 게이미피케이션 + 상태 반응/자율 행동**(P3/P6/P7) — 노션엔 감정 장치 자체가 없음.
  Phase 6에서 mood 반응·유휴 혼잣말·활보 모드까지 방금 구현.
- **개발자 활동 통합**(GitHub 잔디 + Claude Code mtime 프라이버시 수집기, P4/P5) — 노션 무대응.
- **데스크톱 상시 위젯 + Quick Capture**(P5, always-on-top) — 노션은 탭 속 앱.
- **픽셀 오리 오피스**(P16/17) — 노션과 무관한 독자 영역.
